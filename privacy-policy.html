Privacy Policy

At VeriNews, your privacy is a top priority. This policy outlines how we handle and protect your data when using the VeriNews browser extension.

1. Information We Collect

  • Text Selection: We collect only the text you explicitly select and submit via the extension for analysis.
  • Tab URL: We access the current tab’s URL only when you click the extension icon to verify content.

2. How We Use Your Information

  • To send selected text and URL to our AI-powered API for credibility analysis.
  • To return a confidence score and label ("Real" or "Fake") for user interpretation.

3. Data Retention

  • We do not retain or store any personal data, selected text, or URLs after processing.

4. Third-Party Services

  • We use an internal API hosted on our private server with AI models from Hugging Face and TensorFlow.
  • No third-party trackers or analytics are used.

5. Security

  • All communication between the extension and the backend server is encrypted using HTTPS.

6. Your Consent

By using VeriNews, you consent to this privacy policy.

7. Contact Us

If you have any questions about this Privacy Policy, please contact us at salehfree33@gmail.com.
